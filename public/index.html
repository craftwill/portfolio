Description

Ce portfolio à été réalisé à l'aide de visual studio code avec des plugins permettant de travailler en temps réel avec le compilateur de SASS pour le style de la page ainsi qu'un hébergeur local intégré pour la partie serveur. L'arrière plan de la page à été fait avec une librairie nommée P5JS qui permet de facilement dessiner sur un canvas.

La partie Javascript est programmée en modules qui intéragissent entre eux et le serveur est en NodeJS avec des librairies qui sont gérée par le gestionnaire de package npm.

La section 'Projets' vos permet de mieux comprendre qui je suis à l'aide de différent projets sur lesquels j'ai travaillé.
